About Michigan Media Hub

Non-commercial reference directory and live news aggregator

What This Is

Michigan Media Hub is a comprehensive, non-commercial reference cataloging every known media outlet in or serving Michigan as of 2026. Built for journalists, researchers, PR professionals, and curious residents who want one authoritative source for the state's media landscape.
